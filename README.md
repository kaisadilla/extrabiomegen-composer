<h1 align="center">
  Kaisa's Extra Biome Generators's Composer
</h1>

This web app is a tool to author data files for my Minecraft mod
[Kaisa's Extra Biome Generators](https://github.com/kaisadilla/extrabiomegen).

This visual tool also allows analyzing biome map images exported with
the command `/extrabiomegen biomemap <x> <y> <z> <scale> [<filename>]`,
added by the mod, to aid in the construction of new biome sources.
